
1. Time Complexity Analysis of Singly Linked List 

-	PushFront O(1)
-	PopFront O(1)
-	PushBack (no tail) O(n)
-	PopBack (no tail) O(n)
-	PushBack (with tail) O(1)
-	PopBack (with tail) O(n)
